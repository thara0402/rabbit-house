# Vue.js

## Vue CLI 3

### Install
```shell-session
$ npm uninstall -g vue-cli
$ npm install -g @vue/cli
```

### Hello World
```shell-session
$ vue create my-project
$ cd my-project
$ npm run serve
```

### Deploy
```shell-session
$ npm run build
```

### Add Buefy
```shell-session
$ vue add buefy
```


## Nuxt.js

### Install
```shell-session
$ npm install -g @vue/cli-init
```

### Hello World
```shell-session
$ vue init nuxt-community/starter-template my-
$ cd my-nuxt
$ npm install
$ npm run dev
```
